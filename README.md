# YTTrack – Persistência e Modelo


Este repositório contém a implementação das classes do modelo e da persistência do sistema YTTrack, desenvolvidas em Python utilizando SQLite.


## Conteúdo
- Criação do banco de dados
- Classes do modelo (Usuário, Canal, Quadro e Vídeo)
- Classes DAO para persistência
- Código de teste para salvar e recuperar dados


## Tecnologias
- Python
- SQLite
- Programação Orientada a Objetos
